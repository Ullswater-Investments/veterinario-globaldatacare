# Producción Video Whitepaper Técnico - HeyGen

## Objetivo
Crear un video explicativo del Whitepaper Técnico de ACCURO TECHNOLOGY usando HeyGen como plataforma de avatar AI.

## Recursos Disponibles

| Recurso | Ubicación | Descripción |
|---------|-----------|-------------|
| **Componente de presentación** | `/video/whitepaper` | Página optimizada para screenshots 16:9 |
| **Guion completo** | `GUION-HEYGEN.md` | Texto para el avatar por escena |
| **Órdenes de captura** | `ORDENES-CAPTURA.md` | Instrucciones detalladas por screenshot |
| **Notas de producción** | `NOTAS-PRODUCCION.md` | Consejos para HeyGen |
| **Carpeta screenshots** | `screenshots/` | Donde guardar las capturas |

## Flujo de Trabajo

### Paso 1: Acceder al Componente de Video
1. Abre el navegador y navega a: `[URL_DE_TU_APP]/video/whitepaper`
2. Esta ruta es **oculta** (no aparece en la navegación de la app)

### Paso 2: Tomar Screenshots
1. En el componente, activa el **"Modo Screenshot"** (botón en esquina superior derecha)
2. Esto ocultará los controles de navegación para capturas limpias
3. Navega a cada escena usando las flechas del teclado (← →)
4. Para cada escena, toma un screenshot con resolución **1920x1080**
5. Guarda cada captura en `screenshots/` con el nombre indicado

### Paso 3: Preparar HeyGen
1. Sube las 9 imágenes a HeyGen
2. Abre `GUION-HEYGEN.md` y copia el texto de cada escena
3. Aplica efectos Ken Burns suaves donde se indique

### Paso 4: Generar Video
1. Selecciona avatar y voz en español
2. Configura transiciones entre escenas
3. Genera y exporta el video final

## Características del Componente de Video

- **9 escenas** optimizadas para video 16:9
- **Navegación**: Flechas (← →), clicks en botones Anterior/Siguiente
- **Modo Screenshot**: Oculta controles para capturas limpias
- **Toggle Guion**: Muestra/oculta el texto del avatar en cada escena
- **Copiar Guion**: Botón para copiar todo el guion al portapapeles
- **Indicador**: Muestra nombre de archivo sugerido para cada escena
