# Órdenes de Captura Visual

## Instrucciones Generales

1. **Resolución**: 1920x1080 (Full HD, ratio 16:9)
2. **Formato**: PNG (preferido) o JPG de alta calidad
3. **Herramienta**: Usa el modo Screenshot del componente `/video/whitepaper`
4. **Navegador**: Chrome o Firefox en modo pantalla completa (F11)
