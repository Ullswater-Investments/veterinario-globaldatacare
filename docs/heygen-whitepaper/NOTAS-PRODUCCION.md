# Notas de Producción para HeyGen

## Configuración Recomendada de HeyGen

### Avatar
- **Tipo**: Profesional, aspecto ejecutivo/consultor
- **Género**: A elección (preferiblemente que coincida con la voz)
- **Vestimenta**: Formal (traje o camisa profesional)
- **Posición**: De pie o sentado, mirando a cámara

### Voz
- **Idioma**: Español (España)
- **Tono**: Profesional, claro, confiado
- **Velocidad**: Media (ajustar si es necesario)

### Fondo
- **Opción 1**: Usar las imágenes como fondo completo
- **Opción 2**: Avatar superpuesto sobre las imágenes
- **Opción 3**: Picture-in-picture (avatar en esquina)

---

## Efectos Visuales Recomendados

### Efecto Ken Burns (Zoom & Pan)

| Escena | Efecto Sugerido |
|--------|-----------------|
| 01. HERO | Zoom out suave (de cerca a general) |
| 02. MAPA_GRID | Zoom lento hacia el centro del grid |
| 03. CLINICA | Estático o zoom muy suave a la tarjeta |
| 04. PACIENTE | Estático o zoom muy suave a la tarjeta |
| 05. CIRCULAR | Pan suave de arriba a abajo |
| 06. TECH_STACK | Zoom out para mostrar todas las tecnologías |
| 07. METODOLOGIA | Pan vertical siguiendo el timeline |
| 08. INNOVACION | Zoom suave hacia el centro |
| 09. CTA | Zoom in hacia los botones de acción |

### Transiciones Entre Escenas

- **Tipo recomendado**: Crossfade suave (0.5-1 segundo)
- **Evitar**: Transiciones bruscas o efectos llamativos
- **Consistencia**: Usar la misma transición en todo el video

## Timing y Ritmo

### Duración por Escena

| Escena | Palabras | Duración Estimada | Notas |
|--------|----------|-------------------|-------|
| 01 | ~45 | 15-18 seg | Apertura, ritmo medio |
| 02 | ~40 | 15-18 seg | Establecer concepto |
| 03 | ~45 | 18-20 seg | Detalle técnico |
| 04 | ~40 | 15-18 seg | Beneficio claro |
| 05 | ~50 | 20-22 seg | Concepto complejo |
| 06 | ~45 | 18-20 seg | Lista de tecnologías |
| 07 | ~40 | 15-18 seg | Proceso estructurado |
| 08 | ~40 | 15-18 seg | Visión futura |
| 09 | ~35 | 12-15 seg | Cierre con énfasis |

**Duración Total Estimada**: 2:30 - 3:00 minutos

### Pausas Naturales

Insertar pausas breves (0.3-0.5 seg) después de:
- Nombres propios: "ACCURO TECHNOLOGY"
- Conceptos clave: "Zero-Trust", "soberanía", "federado"
- Números y fechas: "1920x1080", "Q2 2025"

## Mejores Prácticas

### Para las Imágenes
1. ✅ Mantener resolución nativa (1920x1080)
2. ✅ Verificar que el texto sea legible
3. ✅ Comprobar contraste con el avatar superpuesto
4. ❌ No comprimir demasiado los PNG/JPG
5. ❌ No recortar bordes importantes

### Para el Audio
1. ✅ Revisar pronunciación de términos técnicos
2. ✅ Añadir música de fondo suave (opcional)
3. ✅ Normalizar volumen entre escenas
4. ❌ No usar música con derechos de autor
5. ❌ No subir volumen de música sobre voz

### Para el Video Final
1. ✅ Exportar en 1080p mínimo (4K si está disponible)
2. ✅ Incluir 2 segundos de negro al inicio y final
3. ✅ Verificar sincronización labios-audio
4. ✅ Revisar en diferentes dispositivos
5. ❌ No comprimir excesivamente

## Elementos Opcionales

### Intro Branding (Opcional)
- Logo ACCURO TECHNOLOGY animado (3-5 segundos antes de Escena 01)
- Música de intro corporativa

### Outro Branding (Opcional)
- Logo final con datos de contacto (5 segundos después de Escena 09)
- URL del sitio web
- Email de contacto

### Subtítulos (Recomendado)
- Generar subtítulos en español
- Opcionalmente en inglés para audiencia internacional
- Estilo: Fondo semitransparente, fuente sans-serif

### Música de Fondo (Opcional)
- Estilo: Corporativo, tecnológico, inspirador
- Volumen: -20dB respecto a la voz
- Fuentes: Artlist, Epidemic Sound, YouTube Audio Library
